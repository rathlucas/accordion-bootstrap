ACCORDION UTILIZANDO BOOTSTRAP

O que é um Accordion?

O Accordion é este elemento com o qual você, o usuário, está interagindo agora! Ele pode ser usado para diversas finalidades, sendo a principal criar uma seção FAQ dentro do seu site.

Como posso criar um Accordion?

Este Accordion em específico foi criado utilizando a biblioteca Bootstrap, combinando cards e a classe collapse. O efeito pode ser feito utilizando HTML, CSS e Javascript puro também, o Bootstrap apenas facilita o processo :D
